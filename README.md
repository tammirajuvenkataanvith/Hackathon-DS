

# 📱 Social Media Friend Suggestion System

## 📌 Overview

This project is a **Social Media Friend Suggestion System** built using core Data Structures without using Java Collection Framework. It manages user records efficiently and simulates real-world friend suggestion logic using optimized algorithms and structures.

---

## 🎯 Features

* Add and manage user records
* Search users by ID (Binary Search)
* Search users by Interest (Linear Search)
* Sort records using multiple algorithms
* Process users using Queue & Circular Queue
* Handle priority users using Heap (Priority Queue)
* Undo operations using Stack
* Fast lookup using Hashing
* Persistent storage using File Handling

---

## 🧩 Data Structures Used

* Linked List → Dynamic storage
* Arrays → Internal processing
* Stack → Undo operations
* Queue → Normal processing
* Circular Queue → Overflow handling
* Priority Queue (Heap) → Urgent processing
* Hash Table → Fast access

---

## ⚙️ Project Modules

1. Registration (Linked List + File Handling)
2. Searching (Binary + Linear Search)
3. Sorting (Bubble, Insertion, Quick Sort)
4. Queue Processing
5. Priority Handling
6. Undo Operation
7. Hashing
8. File Handling

## 🛠️ Technologies Used
